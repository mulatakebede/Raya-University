# Liaison Office Guest House - Guest Satisfaction Survey Analysis 2018

A complete survey analysis tool that generates charts, Excel tables, Word reports, and an interactive HTML dashboard from survey data.

## 🎯 Overview

This tool processes guest satisfaction survey data from the Liaison Office Guest House (2018) and generates:

- **Interactive Web Dashboard** (`dashboard.html`) - Complete report with all charts, tables, and findings
- **4 Data Visualizations** - Mean scores, visit profile, loyalty indicators, distribution charts
- **Excel Analysis** - SPSS-style frequency tables and summary statistics
- **Word Report** - Comprehensive narrative report with recommendations

## 📊 Sample Output

### Dashboard Preview

The dashboard includes:
- Executive Summary with key metrics
- Methodology section
- Respondent Profile (purpose of visit, length of stay)
- Overall Satisfaction analysis
- Visualizations (4 charts)
- Service Area Analysis with summary statistics
- Frequency tables for all questions (interactive tabs)
- Loyalty indicators
- Data-driven recommendations
